For Loops


Loops in Java are utilized when we want to execute a block of code more than once. Java's for loop gives a compact method for composing the design of the loop. The for explanation consumes the instatement, condition, and addition/decrement in one line, giving a more limited loop structure that is simpler to troubleshoot.


Conditions

Less than: a < b
Less than or equal to: a <= b
Greater than: a > b
Greater than or equal to: a >= b
Equal to a == b
Not Equal to: a != b

Conditional Statements

The above conditions can be used with conditional statements to make decisions


The if Statement

Use the if statement to specify a block of Java code to be executed if a condition is true.




The else Statement

Use the else statement to specify a block of code to be executed if the condition is false.




The else if Statement

else if is an extension of if / else, and allows further conditions to be tested
