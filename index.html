Welcome!

Welcome to W5 Universities! Founded in 2022, W5 is a new and up-an-coming programming tutorial site. While currently in early preview mode we aim to have the full site operational soon and can’t wait to bring you even more tutorials in even more languages!


W5 currently only offers tutorials for Java programming – don’t let it discourage you! Most concepts also apply to other C-derived languages such as C++ and C# (Of course with some slight differences in format and keywords).




How to use the site

We’ve tried to make using the W5 site is easy and intuitive as possible! The contents button leads to a detailed list of all the articles and materials present on the site while the articles drop down offers a more direct route to each article.


Material Pages

Each article page includes a written explanation of the concept in question, as well as code examples and tutorial videos when appropriate. Additionally, some pages feature an interactive quiz that tests the knowledge developed by reading the materials.


Accessibility

Located in the footer of the site, you’ll find the accessibility and print features. There are options for changing the font to a more dyslexic friendly format and changing the sites colour theme to a high contrast version. Additionally, the print button can be used to quickly print the contents of whichever page you’re currently visiting. (Please note that currently, only the print feature has been implemented).
